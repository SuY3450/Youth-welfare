## 🌟 프로젝트 소개

청년들이 본인에게 맞는 복지 정책을 찾지 못해 혜택을 놓치는 문제를 해결합니다.  
온통청년, 서울청년몽땅, 경기청년포털 등 **다양한 공공 데이터를 통합**해  
자치구·시군 단위의 지역 밀착형 정보를 AI로 매칭하고 서류 발급처 자동 안내합니다.
