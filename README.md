
# Youth-welfare
> **복지 길잡이 BokjiGPT** — AI 기반 청년 복지 정책 매칭 서비스

2026 캡스톤 프로젝트

---

## 프로젝트 소개

나이, 지역, 소득, 가구 형태 등 개인 조건을 입력하면 AI가 적합한 청년 복지 정책을 매칭하고 신청 서류까지 자동 생성해주는 서비스입니다.  
온통청년, 서울청년몽땅, 경기청년포털 등 다양한 공공 데이터를 통합하여 자치구·시군 단위의 지역 밀착형 정보를 개인 맞춤형으로 제공합니다.
